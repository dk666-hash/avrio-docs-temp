##	Data Catalog

Data Catalog is a centralized repository of data products. It uses metadata for data discoverability and accessibility. It acts as a single source of truth to find and manage data products. You can think of it as an inventory of data. The Data catalog will help you to view Data Products, Data Assets and Business Glossaries.

Data Catalog comprises of the following:

- Data Products
- Data Assets
- Business Glossary
