# Details
Displays the complete details of a data product.

To search for a data product in Search field, type the data product name.

Details tab displays the following metadata under it:

## Name
Displays the name of the data product.

## Business Name

## Description
Displays the description of the data product.

## Data Type
Displays the data product type.

## Tags
Tags help data products with categorization, discovery, and search.
